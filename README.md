Sistema de Gestão de Reservas de Salas
Este projeto foi desenvolvido como parte integrante da disciplina de Projeto Integrador, com o objetivo de criar uma solução funcional para o agendamento e gestão de espaços corporativos. A aplicação permite que usuários autenticados visualizem a disponibilidade de salas, realizem reservas e gerenciem seus próprios agendamentos de forma intuitiva.
