# Sistema de Gestão de Reservas de Salas

Projeto desenvolvido para a disciplina de Projeto Integrador. A aplicação web permite que usuários autenticados visualizem a disponibilidade de salas, realizem agendamentos e gerenciem suas reservas de forma intuitiva.

##  Credenciais de Teste

O sistema possui uma API simulada (Mock DB) no arquivo `api.js` para testes de interface. Utilize os dados abaixo para acessar:

**Usuário de Teste 1**
* **CPF:** `000.000.000-00` (ou apenas `00000000000`)
* **Senha:** `senha1`

**Usuário de Teste 2**
* **CPF:** `111.111.111-11` (ou apenas `11111111111`)
* **Senha:** `senha2`
