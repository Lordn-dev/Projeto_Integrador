# Sistema de Gestão de Reservas de Salas

Projeto desenvolvido para a disciplina de Projeto Integrador. A aplicação web permite que usuários autenticados visualizem a disponibilidade de salas, realizem agendamentos e gerenciem suas reservas de forma intuitiva.
